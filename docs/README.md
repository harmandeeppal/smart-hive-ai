# Documentation Guide

The documentation tree is organised by topic so you can quickly find the right reference:

- `index.md` - global index and navigation tips.
- `core/` - deployment workflows, quick-start checklists, and container build instructions.
- `audio/` - audio ML configuration, pipeline details, dashboard behaviour, and troubleshooting.
- `camera/` - USB camera configuration, diagnostics, and dashboard integration notes.
- `reference/` - configuration catalogues and cross-cutting troubleshooting material.

Historical write-ups and redundant summaries have been removed. When adding new material, place it in the closest topic directory and update `index.md` to keep navigation current.
